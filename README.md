🎉 프로젝트 리팩토링은 코드의 가독성과 유지보수성을 향상시키기 위한 목적입니다. 기존의 코드를 개선하고, 구조를 재정비하며, 성능을 최적화하는 것을 목표로 하였습니다.

특징
✨ 주요 기능:
성능 최적화: 기존의 for 반복문을 Stream API로 변경하여 성능 향상
코드 가독성 향상: Arrays.asList를 사용하여 생성자 가독성 개선 및 주석 추가
